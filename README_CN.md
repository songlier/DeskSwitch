Multi-desktop switcher, Makes one monitor more powerful than multiple monitors

## 使用  
1. 下载 Release 中的 `DeskSwitch.zip`  
2. 解压并运行 `DeskSwitch.exe`  
3. 设定开机自启: 按 `Win + R`, 输入  `shell:startup`, 将 `DeskSwitch.exe` 创建快捷方式到打开的目录  
## 快捷键  
`Ctrl+Alt+Esc`: 切换开启和关闭  
`Ctrl+Alt+Shift+Esc`: 退出程序  
## 触发区域  
通过修改 `area.txt` 可设定触发区域  
其中数值顺序为: {左, 上, 右, 下}  
数值可为数字(px)或屏幕百分比  
## 例外  
部分程序会拦截桌面切换功能导致切换不可用  
可将该程序写入 `except.txt`, 程序将使用 dll 切换桌面  
已知这类程序包括但不限于 `1Remote.exe`, `GameViewer.exe`  
## 设置  
可通过修改 `conf.txt` 以实现自定义  
1. 单双角模式  
设置软件模式为手动切换(双角模式)或自动识别方向(单角模式)  
单角模式:只有左上角为触发区域, 此模式下切换方向为上一个桌面相对当前桌面的方向  
双角模式:左上角与右上角均为触发区域, 切换方向为角对应的左右方向  
2. 单角方向记忆超时  
设定单角模式下多长时间刷新记忆, 该设定时间内视为连续触发, 方向不变, 超过该事件视为切换完成, 下次切换将根据上一个桌面位置调整方向  
3. 连续切换  
开启后鼠标停留在触发角将自动连续切换, 移开将停止, 关闭则单次进入触发角仅触发一次  
4. 连续切换时间间隔  
连续切换开启后的自动切换时间间隔  
5. 自动反向切换  
当前切换方向已是最后一个桌面时, 是否自动反向切换  
6. 离开时触发  
关闭: 鼠标移入触发角则触发切换  
开启: 鼠标移开触发角才触发, 且鼠标在触发角松开(如拖动窗口到角落)和在触发角按下(如点击角落的窗口关闭按钮)不会触发切换桌面  
7. 窗口快捷移动   
当只有 1-2 个桌面时是否开启窗口快捷移动功能  
Alt+Shift+左键: 将当前窗口移动到另一个桌面并切换到该桌面  
Alt+Shift+右键: 将当前窗口移动到另一个桌面  
8. 自动屏蔽右上角  
在使用双角模式时, 当只有 1-2 个桌面时屏蔽右上角, 只有左上角可触发且桌面循环切换  
9. 其它设置  
时间设置, 通常情况下无需调整
